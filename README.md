# alfred-plugins

A collection of [Alfred](https://www.alfredapp.com/) workflows.

## Workflows

### [Claude Session Search](./claude-session-search)

Search through your Claude Code sessions by description (or full conversation
text) and resume the selected one in your terminal — Warp by default, with
`--dangerously-skip-permissions`. Supports iTerm / Terminal / Ghostty, modifier
actions (resume without bypass, open dir, reveal transcript, Quick Look, copy
command) and a configurable editor and flags.

Install: download and double-click
`claude-session-search/Claude Session Search.alfredworkflow`, or see that
folder's [README](./claude-session-search/README.md).
